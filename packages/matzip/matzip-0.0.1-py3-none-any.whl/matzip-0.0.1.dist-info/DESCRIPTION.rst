# matzip

Korean restaurant recommendation library

## Installing

To install matzip, simply:

    $ pip install matzip

Or, you can use:

    $ easy_install matzip

Or, you can also install manually:

    $ git clone https://github.com/symoon94/matzip.git
    $ cd matzip
    $ python setup.py install
