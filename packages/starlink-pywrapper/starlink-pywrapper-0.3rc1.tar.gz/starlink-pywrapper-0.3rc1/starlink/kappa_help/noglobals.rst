

NOGLOBALS
=========


Purpose
~~~~~~~
Resets the KAPPA global parameters


Description
~~~~~~~~~~~
This application resets the KAPPA global parameters, and so makes
their values undefined.


Usage
~~~~~


::

    
       noglobals
