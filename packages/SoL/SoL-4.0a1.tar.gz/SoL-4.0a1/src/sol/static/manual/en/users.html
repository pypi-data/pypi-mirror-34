Users management¶

To interact with the system, that is to be able to insert new tournaments and other entities, one must inserts his credentials into the login panel. Such credentials may be either granted by the administrator, or by a self registration.

Menu actions¶

In addition to the standard actions the menu at the top contains the following items:

Clubs
Opens the management of the clubs which the selected user is responsible of
Championships
Opens the management of the championships which the selected user is responsible of
Players
Opens the management of the players which the selected user is responsible of
Ratings
Opens the management of the ratings which the selected user is responsible of
Tourneys
Opens the management of the tourneys which the selected user is responsible of

Insert and edit¶

All fields, with the exception of language, are mandatory.

Email and password¶

These are the credentials the user shall insert into the login panel. The email must be a valid address and the password may be anything longer than five characters.

Note

While the self registration procedure ascertains the validity of the email address, when it is inserted or modified manually no check is done that the address corresponds to an existing mailbox. It is therefore recommended to pay particular attention: a wrong value precludes the possibility of resetting the password should it be forgotten!

First name and last name¶

These fields cannot be left empty.

Language¶

If set, this overrides the browser’s default when the user logs in.

Owners admin¶

The administrator can grant a owners admin permission to particular users: when a user has this right he can change the ownership of other items (clubs, tournaments…), even of those not belonging to him.

Status¶

The current status of the user:

Registered
the self registration <signin> procedure has not been completed yet: users in this state cannot log in
Confirmed
this is the normal state for an operational user
